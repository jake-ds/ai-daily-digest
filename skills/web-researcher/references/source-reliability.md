# 출처 신뢰도 판단 기준

## 신뢰도 등급

### Tier 1: 최고 신뢰 (가중치 높게 활용)
- 학술 기관: MIT, Stanford, CMU, Berkeley
- 연구 조직: Google Research, DeepMind, OpenAI, Anthropic, Meta AI, Microsoft Research
- 학술 플랫폼: arxiv.org, scholar.google.com
- 공식 블로그: openai.com/blog, anthropic.com/research

### Tier 2: 높은 신뢰
- 주요 테크 미디어: TechCrunch, The Verge, Ars Technica, Wired
- 전문 미디어: MIT Technology Review, IEEE Spectrum
- 벤처/비즈니스: a16z, Sequoia, Y Combinator 블로그
- 데이터 리포트: Statista, Gartner, McKinsey

### Tier 3: 보통 신뢰
- 개발자 블로그: Medium (유명 저자), Dev.to, Hacker News (댓글)
- 커뮤니티: Reddit (r/MachineLearning), Stack Overflow
- 뉴스레터: The Batch, Import AI, TLDR

### Tier 4: 낮은 신뢰 (교차 검증 필요)
- 익명 블로그, 포럼 게시물
- 마케팅 콘텐츠 (제품 홍보 목적)
- 소셜 미디어 (X/Twitter, LinkedIn 개인 포스트)

## 활용 가이드

- Tier 1-2 출처의 수치/인용구는 직접 사용 가능
- Tier 3 출처는 맥락 파악용, 직접 인용 시 출처 명시
- Tier 4 출처는 참고만, 직접 인용 지양
- 같은 정보가 여러 출처에서 확인되면 신뢰도 상승

- facebook.com (로그인 필요)
